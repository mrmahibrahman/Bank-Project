How to use this application: 

1. Go to ATM.java and run the code. 
2. If you want to run the code with the dialog box, invoke the constructor for IOHandlerDialog
3. If you want standard input/output, invoke the constructor for IOHandlerStandard
4. Once you run the code, enter a valid id number that is in the bank, must be numbers only.
5. The code will ask you to make a transaction, and you are presented with 4 choices.
6. If you want to deposit or withdraw, make sure to enter numbers for both the dollar and cents input
7. You could hit exit, "x" button, or cancel, and the program will end. 
8. The output of updated accounts will appear in out_accounts.txt.
